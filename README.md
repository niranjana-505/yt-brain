# YT Brain — AI Video Summarizer

Paste any YouTube link and get an instant summary, key bullet points, and quiz questions powered by Gemini AI.

## Features
- AI-powered video summarization
- Key bullet points extraction
- Auto-generated quiz questions
- Original animated pixel art cat mascot and background
- Responsive design

## Tech Stack
- Python + Flask
- Gemini 2.5 Flash API
- YouTube Data API v3(official Google API)
- HTML + CSS + JavaScript
